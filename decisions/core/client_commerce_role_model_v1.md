定義「客戶類型不只是一個欄位」這件事。

要寫清楚：
	•	client_type = 商業型態
	•	餐廳 / 酒吧 / 零售 / 通路
	•	client_role = 客戶身份
	•	individual / business / distributor / kol / partner / event
	•	acquisition_channel = 來源
	•	line / facebook / website / referral / offline / other

也要寫清楚：
	•	不可用 client_type 取代 client_role
	•	UI 應呈現為兩層，不是混成一個欄位
